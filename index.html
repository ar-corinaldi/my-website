Some Important Courses

Algorithmic and Object Oriented Programming

The Algorithmic and Object Oriented Programming 1 (APO1) course is a course programming introductory, in which students who have never programmed they obtain the necessary skills to develop complete applications with interfaces graphics, persistence and automated tests.

Data Structure

Data structures, is the last course of the basic programming cycle, the purpose of this course is to generate in the student the ability to design and implement the data structures in main memory necessary to solve a problem, taking into account the quality criteria efficiency in runtime and main memory of the solution.

Design and Analysis Algorithms

Present basic design concepts and algorithm analysis. At the end of the course, the student must be able to apply algorithm development techniques such as divide and conquer, dynamic programming, use various search algorithms and analyze their complexity in time and space.

Software Architecture

Develop the competencies of defining, justifying, implementing and testing a software architecture for a complex system. The architecture is built in a team using a development methodology that includes agile practices (e.g., of SCRUM).

Software Development

The purpose of this course is to develop skills in the areas of software design and architecture, relying on methodological tools such as architecture styles, patterns (e.g. design, architecture). Likewise, it is intended to develop the ability to use and understand the impact of technology on software architecture.

Transactional Systems

The course studies the problem of persistence and shared access of large volumes of data in the context of the design, development and operation of transactional systems. It addresses both the methodological aspects of design, such as transactional and relational database technology.
